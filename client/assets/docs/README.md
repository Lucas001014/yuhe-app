## 📋 快速开始

### Android 上架流程

```bash
# 1. 阅读 Android 上架指南
docs/ANDROID_UPLOAD_GUIDE.md

# 2. 注册 Google Play 开发者账号
# 访问：https://play.google.com/console

# 3. 准备应用素材
# - 应用图标：assets/images/icon.png（已完成）
# - 应用截图：按照 screenshot-guide.md 拍摄
# - 应用描述：使用 app-store-description.md 中的文案

# 4. 构建应用 AAB 文件
cd /workspace/projects/client
npx eas build --platform android --profile production

# 5. 下载 AAB 文件
# 构建完成后，从 EAS 控制台下载

# 6. 上传到 Google Play Console
# 登录：https://play.google.com/console
# 按照 checklist.md 逐项完成配置

# 7. 提交审核
# 填写测试账号（test-accounts.md）
# 提交审核
```

### iOS 上架流程

```bash
# 1. 注册 Apple Developer Program
# 访问：https://developer.apple.com/programs/

# 2. 准备应用素材
# - 应用图标：assets/images/icon.png（已完成）
# - 应用截图：按照 screenshot-guide.md 拍摄（iOS 尺寸）
# - 应用描述：使用 app-store-description.md 中的文案

# 3. 配置 iOS 签名
# 在 Apple Developer 账号中创建证书和 Provisioning Profile

# 4. 构建应用 IPA 文件
cd /workspace/projects/client
npx eas build --platform ios --profile production

# 5. 下载 IPA 文件
# 构建完成后，从 EAS 控制台下载

# 6. 上传到 App Store Connect
# 登录：https://appstoreconnect.apple.com
# 按照 checklist.md 逐项完成配置

# 7. 提交审核
# 填写测试账号（test-accounts.md）
# 提交审核
```

## 📌 注意事项

### ⚠️ 重要提醒

1. **隐私政策和用户协议必须部署到官网**
   - 隐私政策：https://www.yuhe.app/privacy
   - 用户协议：https://www.yuhe.app/terms
