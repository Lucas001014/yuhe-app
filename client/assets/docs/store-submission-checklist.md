## 二、应用素材准备

### 应用图标

- [ ] 制作 512x512px 应用图标（PNG）

- [ ] 准备应用背景色（#1677FF）
- [ ] 图标符合平台规范（Android/iOS）
- [ ] 图标无水印、无其他应用标识

### 应用截图

#### Android 截图
- [ ] 准备至少 2 张手机截图
- [ ] 准备最多 8 张截图
- [ ] 尺寸：1080x1920px 或更高
- [ ] 格式：PNG 或 JPG
- [ ] 内容：首页、发帖、合作对接、个人中心等
- [ ] 无加载中的占位符
- [ ] 无系统状态栏（或已裁剪）
- [ ] 文件大小 < 5MB

#### iOS 截图
- [ ] 准备至少 2 张截图
- [ ] 准备最多 10 张截图
- [ ] 尺寸：1242x2208px（iPhone 6.7"）
- [ ] 格式：PNG 或 JPG
- [ ] 内容：首页、发帖、合作对接、个人中心等
- [ ] 无加载中的占位符
- [ ] 无系统状态栏（或已裁剪）
- [ ] 文件大小 < 5MB

### 功能图形

- [ ] 制作 512x512px 功能图形（PNG，透明背景）
- [ ] 设计简洁，突出品牌特色

### 宣传视频（可选）

- [ ] 录制 15-30 秒演示视频
- [ ] 格式：MP4 或 MOV

## 重要提示

### ⚠️ 不可忽视的事项

1. **签名密钥管理**
   - Android 签名密钥必须妥善保管
   - iOS 证书和 Provisioning Profile 必须配置正确
   - 密钥丢失将无法更新应用

2. **隐私政策**
   - 隐私政策必须真实有效
   - 隐私政策 URL 必须可访问
   - 隐私政策内容必须符合法律法规

3. **测试账号**
   - 测试账号信息必须正确
   - 测试账号功能必须完整
   - 测试账号必须上传到应用商店

4. **应用截图**
   - 截图必须展示真实应用界面
   - 截图不能包含其他应用的标识
   - 截图不能包含违规内容

5. **内容审核**
   - 应用内容必须符合平台政策
   - 不能包含色情、暴力、赌博等违规内容
   - 不能诱导用户评分或评价
