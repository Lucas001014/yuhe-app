# 控制面板使用说明

## 访问方式

### 通过浏览器访问
1. 打开浏览器，访问：`http://localhost:5000`
2. 在首页右上角，点击 **紫色滑块图标** ⚙️ 进入控制面板

### 通过应用访问
1. 打开应用，进入首页
2. 在首页右上角，点击 **紫色滑块图标** 进入控制面板

## 功能说明

### 1. 仪表盘
显示应用的核心数据和快速操作入口：
- **帖子总数**：显示已发布的帖子数量
- **用户总数**：显示注册用户数量
- **类别数量**：显示内容分类数量
- **运行时间**：显示服务器运行时长（分钟）

**快速操作按钮**：
- 🟦 **管理后台**：跳转到管理后台页面
- 🟩 **我的钱包**：跳转到钱包页面，查看余额和提现
- 🟨 **返回首页**：返回应用首页
- 🟪 **刷新面板**：刷新控制面板数据

### 2. 内容管理
提供内容管理的快速入口：
- 📄 **帖子管理**：管理所有帖子内容
- 👥 **用户管理**：管理用户和权限
- 🏷️ **类别管理**：管理内容分类

### 3. 前端控制
提供前端调试和管理功能：

#### 缓存管理
- **清理缓存**：清理应用缓存数据
  - Web 端：清理 localStorage 和 sessionStorage
  - 移动端：使用 AsyncStorage 清理缓存

#### 页面操作
- **刷新页面**：刷新当前页面
  - Web 端：直接刷新页面
  - 移动端：提示用户使用下拉刷新

#### 命令执行
支持以下命令：
- `clear`：清理系统日志
- `reload`：刷新仪表盘数据
- `version`：显示应用版本信息

### 4. 系统日志
实时显示操作日志：
- **INFO**：普通信息（蓝色）
- **SUCCESS**：成功信息（绿色）
- **WARN**：警告信息（黄色）
- **ERROR**：错误信息（红色）

日志最多保留 100 条，自动滚动到最新日志。

## 使用技巧

### 快速切换页面
使用控制面板的快速操作按钮，可以快速跳转到各个功能页面，无需通过底部导航栏多次点击。

### 调试前端问题
遇到前端问题时，可以使用"前端控制"功能：
1. 清理缓存后刷新页面
2. 使用命令行执行特定操作
3. 查看系统日志了解错误信息

### 查看系统状态
通过仪表盘快速了解应用的整体运行状态，包括内容数量、用户数量和服务器运行时间。

## 常见问题

### Q: 为什么底部导航栏不见了？
A: 确保通过首页访问，而不是直接访问 `/index` 路由。首页已经配置了底部导航栏。

### Q: 控制面板的按钮没有反应？
A: 确保后端服务正常运行（端口 9091）。可以通过访问 `/api/v1/health` 检查服务状态。

### Q: 如何返回首页？
A: 点击控制面板的"返回首页"按钮，或者点击底部导航栏的"首页"标签。

### Q: 移动端和 Web 端功能有什么区别？
A: 大部分功能相同，但部分功能有差异：
- 缓存管理：Web 端使用 localStorage，移动端使用 AsyncStorage
- 页面刷新：Web 端可以直接刷新，移动端需要使用下拉刷新

## 技术实现

- **路由**：使用 Expo Router 的 Stack 导航
- **样式**：使用主题驱动的样式系统
- **状态管理**：使用 React Hooks
- **跨平台**：支持 iOS、Android 和 Web 三端
